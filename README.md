## 🎯 هدف از این پروژه / Objectives of this project  

| فارسی | English |
|-------|---------|
| 1️⃣ آشنایی اولیه با Git و GitHub – اولین تجربه واقعی در تیم‌کاری بر روی یک مخزن مشترک. | 1️⃣ Introduction to Git & GitHub – your first real team experience on a shared repository. |
| 2️⃣ مستندسازی همکاری‌ها – ذخیره اطلاعات هر عضو به صورت ساختارمند، تا بتوانید نقش‌ها و دستاوردهایشان را در آینده مشاهده کنید. | 2️⃣ Documenting collaboration – structured storage of each member’s details for future reference. |
| 3️⃣ شفافیت و نمایش کار گروهی – همه اعضا نام، مشخصات و لینک GitHub خود را در `README.md` قرار می‌دهند؛ این باعث ایجاد حس مسئولیت‌پذیری و شفافیت می‌شود. | 3️⃣ Transparency & teamwork showcase – every member lists their name, details and a link to their GitHub in the `README.md`, fostering accountability. |
