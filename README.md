## 🎯 هدف و انگیزه / Objectives & Motivation  

| فارسی | English |
|-------|---------|
| 1️⃣ **آشنایی اولیه با Git و GitHub** – اولین تجربهٔ واقعی در تیم‌کاری بر روی یک مخزن مشترک، برای دانشجویان مقطع کارشناسی دانشگاه چمران. | 1️⃣ **Introduction to Git & GitHub** – your first real team experience on a shared repository, specifically for undergraduate students at Chamran University. |
| 2️⃣ **مستندسازی همکاری‌ها** – ذخیره اطلاعات هر عضو به صورت ساختارمند، تا بتوانید نقش‌ها و دستاوردهایشان را در آینده مشاهده کنید. | 2️⃣ **Documenting collaboration** – structured storage of each member’s details so that roles and achievements can be tracked later. |
| 3️⃣ **شفافیت و نمایش کار گروهی** – همه اعضا نام، مشخصات و لینک GitHub خود را در `README.md` قرار می‌دهند؛ این باعث ایجاد حس مسئولیت‌پذیری و شفافیت می‌شود. | 3️⃣ **Transparency & teamwork showcase** – every member lists their name, details and a link to their GitHub in the `README.md`, fostering accountability. |
| 4️⃣ **تقویت مهارت‌های نرم‌افزاری** – آشنایی با شاخه‌بندی (branching)، Pull Requests، merge conflict و CI/CD پایه. | 4️⃣ **Strengthening software skills** – familiarity with branching, pull requests, merge conflicts and basic CI/CD workflows. |
