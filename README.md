### 🎯 هدف از این پروژه  
1. **آشنایی اولیه با Git و GitHub** – اولین تجربهٔ واقعی در تیم‌کاری بر روی یک مخزن مشترک.  
2. **مستندسازی همکاری‌ها** – ذخیره اطلاعات هر عضو به صورت ساختارمند، تا بتوانید در آینده به راحتی نقش‌ها و دستاوردهایشان را مشاهده کنید.  
3. **شفافیت و نمایش کار گروهی** – تمام اعضا در فایل `README.md` نام، مشخصات و لینک GitHub خود را قرار می‌دهند؛ این باعث ایجاد حس مسئولیت‌پذیری و شفافیت در پروژه می‌شود.
